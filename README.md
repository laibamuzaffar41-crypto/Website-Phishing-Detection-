# Phishing Website Detection using Machine Learning

# Overview
This project detects phishing websites using machine learning techniques.

# Objective
To classify websites as phishing or legitimate based on various features.

# Dataset
Phishing Website Dataset (from Kaggle)

# Technologies Used
- Python
- Pandas
- Scikit-learn

# Models Used
- Logistic Regression
- Decision Tree

# Results
- Logistic Regression Accuracy: 98.85%
- Decision Tree Accuracy: 100% (possible overfitting)

# Conclusion
Machine learning models can effectively detect phishing websites and improve cybersecurity.
